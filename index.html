jQuery - tabs1.2

What is it ?

This is another plugin to manage some tabs. By default this plugin propose something very basic, just like this : jQueryUI - tabs.

But the amazing here is if you also include the plugin jquery-dragndrop in your HTML, you'll be able to reorder the tabs as you want with your mouse.

Needs

This plugin need 3 or 4 files :

  • jquery.js
  • jquery-tabs.js
  • jquery-tabs.css
  • jquery-dragndrop.js optional (only if you want to allow the move of the tabs)
